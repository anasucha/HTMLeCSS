Principais Formatações

Negrito/Destaque

A forma semântica de se colocar um termo em negrito é dizer que ele está em destaque. Para isso, utilizamos a tag STRONG

Itálico/Ênfase

A forma de indicar que o termo está em ênfase é com a tag EM. Olha como fica enfatizado

Marcador de texto

Para marcar um texto, num efeito grifo, é só usar a tag MARK e para escolher a cor, eu posso usar o css como exemplo: vou usar esse texto

Texto grande e pequeno

Para usar texto grande, usa-se a TAG big - que está obsoleta

Para usar texto pequeno, é só usar a TAG small - que ainda é usada

Texto Deletado

Para riscar um texto, é só utilizar a tag DEL. Exemplificando aqui.

Texto inserido ou Sublinhado

Para sublinhar, ou indicar que um texto foi adicionado depois, é so utilizar a tag INS. Pra ter como referência.

Texto sobrescrito

Só usar a TAG sup. Ex: x3

Texto subscrito

Só usar a TAG sub. Ex: H2O
